Ejercicio #2: PROMEDIO DE 3 NÚMEROS

Escriba un algoritmo que pregunte por 3 números y muestre como resultado el promedio de estos.

NOMBRE DE LA VARIABLE

  • a
  • b
  • c
  • R

DESCRIPCIÓN

  • Dame el primer número
  • Dame el segundo número
  • Dame el tercer número
  • Resultado del PROMEDIO

TIPO

  • Float
  • Float
  • Float
  • Float
  1. Inicio
  2. Leer: a, b, c,
  3. Hacer: R = ((parseFloat(a) + parseFloat(b) + parseFloat(c)) / 3)
  4. Mostrar resultado: R
  5. Fin
